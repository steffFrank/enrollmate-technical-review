# Review Scope

This extract is centered on the Master project requirements:

- MPA application structure
- RESTful authentication APIs
- Register/login/logout frontend
- Session-aware profile area
- User profile update functionality
- Shared Server Action helper
- Focused client state with Zustand
- Relevant tests
- Complete documentation

The complete EnrollMate product also includes course recommendations, admissions workflow, widget embedding, analytics, evaluation, billing, and integrations. Those areas are documented but only selectively included here when they support the academic review.
